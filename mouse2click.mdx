---
title: "mouse2click"
---

Simulates a full right mouse button click.

```luau
mouse2click(): ()
```

## Returns

<code title="the function does not return a value.">void</code>

## Example

<CodeGroup>

```luau Example 1
-- mouse2click(): ()
```

```luau Example 2
mouse2click() -- Performs right click
```

</CodeGroup>

<Tip>
  Fires a right mouse button press and release.
</Tip>
