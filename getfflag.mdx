```luau
getfflag(name: string): any
```

Pass the flag's literal name, without a prefix (e.g. `"EnablePreloading"{:luau}`, not `"DFFlagEnablePreloading"{:luau}`).

Flag names follow one of six type prefixes: `FFlag{:luau}` / `DFFlag{:luau}` (booleans), `FInt{:luau}` / `DFInt{:luau}` (integers), `FString{:luau}` / `DFString{:luau}` (strings). The leading `D` marks dynamic flags, which refresh at runtime; the rest lock at session start.

## Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `name` | `string` | Flag name. |

## Returns

Return type tracks the flag's declared type:

| Flag prefix | Returns |
|-------------|---------|
| `FFlag{:luau}` or `DFFlag{:luau}` | `boolean` |
| `FInt{:luau}` or `DFInt{:luau}` | `number` |
| `FString{:luau}` or `DFString{:luau}` | `string` |
