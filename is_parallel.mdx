```luau
is_parallel(): boolean
```

Roblox runs most code on one serial thread. Code inside an [Actor](https://create.roblox.com/docs/reference/engine/classes/Actor) that has called `task.desynchronize(){:luau}` runs in parallel instead - it is desynchronized. `task.synchronize(){:luau}` brings that thread back to serial. `is_parallel(){:luau}` reports which side the thread that calls it is on right now.

A parallel thread can read values from the game, but it cannot change <abbr title="TODO link user to thread-safe properties">some of them</abbr>. Call `is_parallel(){:luau}` before a write and run `task.synchronize(){:luau}` first if it returns `true{:luau}`.

## Returns

`boolean` - `true{:luau}` if the calling thread is currently parallel, `false{:luau}` if it is serial.
