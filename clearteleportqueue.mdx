---
title: "clearteleportqueue"
---

Clears every script queued with `queueonteleport` so none of them run on the next teleport.

```luau
clearteleportqueue(): ()
```

## Returns

<code title="the function does not return a value.">void</code>

## Aliases

- `clear_teleport_queue`
- `clearqueueonteleport`
