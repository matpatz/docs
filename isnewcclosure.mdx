---
title: "isnewcclosure"
---

Checks whether the specified function is a C closure wrapper generated by the `newcclosure` function.

```luau
isnewcclosure(func: function): boolean
```

## Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| `func` | `function` | The function to inspect. |

## Returns

`boolean` - `true` if the function was created via `newcclosure`, `false` otherwise.

## Example

<CodeGroup>

```luau Example 1
local function test()
    return true
end

local cclosure = newcclosure(test)
assert(isnewcclosure(cclosure) == true, "Should identify as a new C closure wrapper")
assert(isnewcclosure(test) == false, "The original Lua closure is not a new C closure")
```

</CodeGroup>
