---
title: "is_parallel"
---

```luau
function is_parallel(): boolean
```

 Returns whether the current thread is running in parallel

> A thread is parallel when it belongs to an Actor: meaning it runs concurrently alongside other actor threads instead of on the main serial thread.

<CodeGroup>

```luau Example 1
print(is_parallel()) -- false (main thread)
```

```luau Example 2
run_on_actor(getactors()[1], [[
    print(is_parallel()) -- true (actor thread)
]])
```

```text Example 3
run_on_thread(getactorthreads()[1], [[
	print(is_parallel()) -- false
]])
```

</CodeGroup>
