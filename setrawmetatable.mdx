title: "setrawmetatable"

```luau
setrawmetatable(object: any, metatable: table?): any
```

If `object{:luau}`'s current metatable has `__metatable{:luau}` set, `setmetatable{:luau}` errors out. <br /> `setrawmetatable{:luau}` skips that lock and writes the metatable directly. Pass `nil{:luau}` for `metatable{:luau}` to clear `object{:luau}`'s metatable instead.

Most commonly used to replace the metatable of an Instance whose `__metatable{:luau}` field would otherwise block a normal `setmetatable{:luau}` call.

## Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `object` | `any` | The value to attach the metatable to. |
| `metatable` | `table?` | The metatable to write. |

## Returns

`any` - the same `object{:luau}` passed in, by reference. Useful for chaining: `local t = setrawmetatable({}, mt){:luau}`.
