---
title: "crypt.hmac"
---

Computes a Hash-based Message Authentication Code (HMAC) for the given data payload using a secret key and hashing algorithm.

```luau
crypt.hmac(data: string, key: string, algorithm: string): string
```

## Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| `data` | `string` | The data message payload to sign. |
| `key` | `string` | The secret key to authenticate the message. |
| `algorithm` | `string` | The hashing algorithm to use (e.g. `'md5'`, `'sha1'`, `'sha256'`, `'sha512'`). |

## Returns

`string` - The resulting HMAC signature as a hexadecimal string.

## Example

<CodeGroup>

```luau Example 1
local key = "secret-key"
local data = "important-message-payload"
local signature = crypt.hmac(data, key, "sha256")
print("HMAC Signature: " .. signature)
```

</CodeGroup>
