---
title: "run_on_actor"
---

```luau
function run_on_actor(actor: Actor, script: string, ...): ()
```

Executes code on an actor's thread.

> The actor must be alive, if the actor is destroyed its thread is gone.

<CodeGroup>

```luau Example 1
local actor = getactors()[1]

run_on_actor(actor, [[
    print(`Hello from {actor}!`) 
	print(#getgenv()) -- 0: getgenv is cleared

	local actor = getactors()[1]
	print(getsenv(actor:FindFirstChildWhichIsA("ModuleScript"))._G)  -- table
]])
```

```luau Example 2
local actors = getactors()
table.foreach(actors, function(i, actor)
    run_on_actor(actor, [[
        print(actor) -- nil
    ]])
end)
```

</CodeGroup>
