```luau
isexecutorclosure(func: function): boolean
```

Most commonly used to filter the `getgc(){:luau}`/`filtergc(){:luau}` output down to functions your script has created.

The check looks at where the closure was made, not which script it came from originally. <br /> A function *pulled* from a Roblox script's environment with `getsenv(){:luau}` keeps its Roblox origin, and returns `false{:luau}`.<br /> A function *produced* from that same script with `getscriptclosure(){:luau}` returns `true{:luau}` instead.

## Parameters

| Parameter | Type       | Description |
|-----------|------------|-------------|
| `func`    | `function` | The function to check. |

## Returns

`true{:luau}` if `func{:luau}` was created by the executor.<br /> `false{:luau}` if the Roblox game has created it.

## Aliases

- `checkclosure{:luau}`
- `isourclosure{:luau}`

## Example

In a Roblox game, use `getgc(){:luau}` and split the function results into executor-allocated vs Roblox-built.

## Example

<CodeGroup>

```luau Example 1
isexecutorclosure(func: function): boolean
```

```luau Example 2
local mine, theirs = 0, 0
for _, fn in ipairs(getgc()) do
    if type(fn) == "function" then
        if isexecutorclosure(fn) then
            mine += 1
        else
            theirs += 1
        end
    end
end

print(`executor-allocated: {mine}, Roblox-allocated: {theirs}`)
```
