```luau
getconnections(signal: RBXScriptSignal): {Connection}
```

Each entry in the returned table is a `Connection{:luau}` object - one per live `:Connect{:luau}` call on `signal{:luau}` at the moment you call. See [`Connection`](/docs/signals/connection) for the object's fields and methods.

Most commonly used to break anti-AFK by disabling every handler on `Player.Idled{:luau}`.

## Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `signal` | [`RBXScriptSignal`](https://create.roblox.com/docs/reference/engine/datatypes/RBXScriptSignal) | The signal whose connections to enumerate. |

## Returns

[`{Connection}`](/docs/signals/connection) - an array of wrappers, one per live `:Connect{:luau}` call on `signal{:luau}`. Foreign-state connections are included; their `Function{:luau}` and `Thread{:luau}` fields read as `nil{:luau}` but the methods still work.

## Example

<CodeGroup>

```luau Example 1

```

```luau Example 2
local LocalPlayer = game.Players.LocalPlayer

for _, conn in ipairs(getconnections(LocalPlayer.Idled)) do
    conn:Disable()
end
