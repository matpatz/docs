```luau
crypt.base64decode(data: string): string
```

Reverses `crypt.base64encode{:luau}`. Give it valid Base64 and you get back exactly the original data, byte for byte.

## Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `data` | `string` | The Base64 text to decode. |

## Returns

`string` - the decoded bytes.

- `base64_decode{:luau}`
- `base64decode{:luau}`
- `crypt.base64.decode{:luau}`
