---
title: "mouse1click"
---

Simulates a full left mouse button click.

```luau
mouse1click(): ()
```

## Returns

<code title="the function does not return a value.">void</code>

## Example

<CodeGroup>

```luau Example 1
mouse1click()
```

```luau Example 2
mouse1click() -- Performs left click
```

</CodeGroup>

<Tip>
  Simulates both press and release events.
</Tip>
