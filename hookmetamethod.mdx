title: "hookmetamethod"

```luau
hookmetamethod(object: any, method: string, hook: function): function
```

`game{:luau}`'s metatable is locked - unlock it with `setreadonly{:luau}` before you hook.

Most commonly used to hook `game{:luau}`'s `__namecall{:luau}` metamethod and watch every method call the game makes, which is how a remote spy or an anti-kick works. Other metamethods come up too - `__index{:luau}`, `__newindex{:luau}`, `__tostring{:luau}`, `__eq{:luau}`, `__call{:luau}`, etc.

<Aside type="danger">
Always call the **returned handle** from inside your hook, never re-call the method on `object{:luau}`. For a `__namecall{:luau}` hook, calling `self:Method(){:luau}` again triggers the same metamethod and recurses until the client crashes. The handle skips the hook and runs the original metamethod - `return old(self, ...){:luau}`.
</Aside>

## Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `object`  | `any` | The object whose metatable is hooked (table, Instance, or userdata). |
| `method`  | `string` | Name of the metamethod to hook, e.g. `"__namecall"{:luau}`. |
| `hook`    | `function` | The function that runs in the metamethod's place. |

## Returns

`function` - the original metamethod, captured before the hook was applied. Call it from inside `hook{:luau}` to run the original behavior - `return original(self, ...){:luau}` for a `__namecall{:luau}` hook.

Keep this handle in a local. Calling it is the only safe way to reach the original metamethod once the hook is live.

## Aliases

- `hookmeta{:luau}`

```luau

## Example

<CodeGroup>

```luau Example 1
hookmetamethod(object: any, method: string, hook: function): function
```

```luau Example 2
local player = game:GetService("Players").LocalPlayer
local mt = getrawmetatable(game)
setreadonly(mt, false)

local oldNamecall
oldNamecall = hookmetamethod(game, "__namecall", function(self, ...)
    -- put a pcall here - so if your code errors, the game won't crash
    local s, isKick = pcall(function()
        return string.lower(getnamecallmethod()) == "kick" and self == player
    end)

    if s and isKick then
        warn(`Client tried to kick!`)
        return
    end

    return oldNamecall(self, ...)
end)

-- This only stops a client-sided :Kick(). A server kick still goes through.
```

</CodeGroup>
