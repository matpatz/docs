Swaps the `method` metamethod on `object`'s metatable for `hook`, and returns the original metamethod.

```luau
hookmetamethod(object: any, method: string, hook: function): function
```

`game`'s metatable is locked - unlock it with `setreadonly` before you hook.

Most commonly used to hook `game`'s `__namecall` metamethod and watch every method call the game makes, which is how a remote spy or an anti-kick works. Other metamethods come up too - `__index`, `__newindex`, `__tostring`, `__eq`, `__call`, etc.

<Warning>
  Always call the **returned handle** from inside your hook, never re-call the method on `object`. For a `__namecall` hook, calling `self:Method()` again triggers the same metamethod and recurses until the client crashes. The handle skips the hook and runs the original metamethod - `return old(self, ...)`.
</Warning>

## Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `object`  | `any` | The object whose metatable is hooked (table, Instance, or userdata). |
| `method`  | `string` | Name of the metamethod to hook, e.g. `"__namecall"`. |
| `hook`    | `function` | The function that runs in the metamethod's place. |

## Returns

`function` - the original metamethod, captured before the hook was applied. Call it from inside `hook` to run the original behavior - `return original(self, ...)` for a `__namecall` hook.

Keep this handle in a local. Calling it is the only safe way to reach the original metamethod once the hook is live.

## Aliases

- `hookmeta`

## Example

In a Roblox game, a LocalScript or ModuleScript can call `:Kick()` on the `LocalPlayer` to boot you out. Catch that `Kick` namecall and stop it before it runs.

<Aside type="note">
Always wrap your logic in `pcall`: if your code (inside the hook) errors, the namecall returns nothing - same break as a bare `return`.

<CodeGroup>

```luau Example 1
local oldNamecall = hookmetamethod(game, "__namecall", function(self, ...)
    return oldNamecall(self, ...)
end)
```

```luau Example 2
local player = game:GetService("Players").LocalPlayer
local mt = getrawmetatable(game)
setreadonly(mt, false)

local oldNamecall
oldNamecall = hookmetamethod(game, "__namecall", function(self, ...)
    -- put a pcall here - so if your code errors, the game won't crash
    local s, isKick = pcall(function()
        return string.lower(getnamecallmethod()) == "kick" and self == player
    end)

    if s and isKick then
        warn(`Client tried to kick!`)
        return
    end

    return oldNamecall(self, ...)
end)

-- This only stops a client-sided :Kick(). A server kick still goes through.
```

</CodeGroup>

  Always wrap your logic in `pcall`: if your code (inside the hook) errors, the namecall returns nothing - same break as a bare `return`.
</Tip>
