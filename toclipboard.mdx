title: "toclipboard"

```luau
toclipboard(text: string): ()
```

The resulting clipboard contents start with a <abbr title="ASCII byte 13. Historically moved a typewriter or terminal cursor back to the start of a line.">carriage return byte</abbr> (`0x0D{:luau}`), then the input string. Not equivalent to `setclipboard{:luau}`, which writes a `string{:luau}` argument without any prefix.

## Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `text` | `string` | The text written after the carriage return prefix. |

## Returns

<code title="the function does not return a value.">void</code>

## Aliases

- `write_clipboard{:luau}`

## Example
```
