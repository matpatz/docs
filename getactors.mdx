---
title: "getactors"
---

```luau
function getactors(): {Actor}
```

<Tip>
  Alias: get\_actors
</Tip>

**Example:**

<CodeGroup>

```luau Example 1 highlight={1}
local actors = getactors()
table.foreachi(actors, function(i, actor)
    print(i, actor.Name)
end)
```

```luau Example 2 highlight={1}
local actor = getactors()[1]
run_on_actor(actor, [[
	print(`Hello from {actor}`)
]])
```

```luau Example 3
print(table.find(getactors(), "Client"))
```

</CodeGroup>

task.spawn(function()
