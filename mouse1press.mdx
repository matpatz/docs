title: "mouse1press"

Simulates holding down the left mouse button.

```luau
mouse1press()
task.wait(0.5)
mouse1release()
```

## Returns

<code title="the function does not return a value.">void</code>

## Aliases

- `Mouse1Press`

## Example

Hold the left mouse button down for a fixed duration, then release it after a short while.
