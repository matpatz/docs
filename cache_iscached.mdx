title: "cache.iscached"
---

```luau
cache.iscached(instance: Instance): boolean
```

Seliware keeps an internal cache of game Instances; an Instance you reach through the game tree is normally tracked there, so `cache.iscached{:luau}` returns `true{:luau}` for it. Once `cache.invalidate{:luau}` removes an Instance from that cache, `cache.iscached{:luau}` returns `false{:luau}` for it.

Most commonly used to confirm a reference was actually detached after an invalidate call.

## Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `instance` | [`Instance`](https://create.roblox.com/docs/reference/engine/classes/Instance) | The Instance to look up in the cache. |

## Returns

`true{:luau}` if `instance{:luau}` is tracked in the instance cache, `false{:luau}` if it is not.

## Example

<CodeGroup>

```luau Example 1
cache.iscached(instance: Instance): boolean
```

```luau Example 2
local remote = game:GetService("ReplicatedStorage"):FindFirstChild("RemoteEvent")

print(cache.iscached(remote)) -- expected true: reached through the game tree

cache.invalidate(remote)

if not cache.iscached(remote) then
    print("dropped from the cache - the next fetch is a new value")
end
```

</CodeGroup>

<Tip>
