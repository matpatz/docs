```luau
compareinstances(a: Instance, b: Instance): boolean
```

Calling `cloneref{:luau}` on an Instance gives you a separate Lua reference. Roblox treats both as the same Instance, but using the `=={:luau}` operator compares the objects themselves - so `original == cloneref(original){:luau}` will return `false{:luau}`. `compareinstances{:luau}` checks the Roblox Instances behind them instead.

Most commonly used after `cloneref{:luau}` to confirm a clone and its original are the same Roblox Instance.

## Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `a` | `Instance` | The first instance to compare. |
| `b` | `Instance` | The second instance to compare. |

## Returns

`boolean` - `true{:luau}` when both arguments are the same Roblox Instance; otherwise `false{:luau}`.

## Aliases

- `checkinst{:luau}`

## Example

<CodeGroup>

```luau Example 1
compareinstances(a: Instance, b: Instance): boolean
```

```luau Example 2
local players = game:GetService("Players")
local cleanPlayers = cloneref(players)

print(players == cleanPlayers)                  --> false
print(compareinstances(players, cleanPlayers))  --> true
```

</CodeGroup>

<Tip>
  Bypasses standard Lua `==` reference inequality for cloned references.
</Tip>
