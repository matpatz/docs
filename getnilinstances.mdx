```luau
getnilinstances(): {Instance}
```

These are usually Instances a script created with `Instance.new{:luau}` and never gave a parent, or Instances the game detached from `workspace{:luau}` but a reference is still holding onto.

Every Instance `getnilinstances{:luau}` returns is also in `getinstances(){:luau}`'s result - `getnilinstances{:luau}` is the floating-in-memory slice of the same set.

## Returns

`{Instance}{:luau}` - an array of Instances with no spot in the game hierarchy. Order is not guaranteed.

end
```
