```luau
readfile(path: string): string
```

`path{:luau}` is workspace-relative. There's no encoding step - you get the exact bytes that are on disk, so binary files read fine too.

## Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `path` | `string` | File to read. |

## Returns

`string` - raw contents of the file. The string can contain any byte value, including embedded `"\0"{:luau}` and binary data.
