---
title: "cloneref"
---

Creates a new, unique reference wrapper pointing to the specified Roblox instance. This reference is not equal to standard references in Lua comparisons (`==`), but points to the same underlying game object.

```luau
cloneref(instance: Instance): Instance
```

## Parameters

| Parameter | Type | Description |
| --- | --- | --- |
| `instance` | [`Instance`](https://create.roblox.com/docs/reference/engine/classes/Instance) | The Roblox instance to create a new reference wrapper for. |

## Returns

`Instance` - A new, unique reference pointing to the same instance.

## Example

<CodeGroup>

```luau Example 1
local part = Instance.new("Part")
local clone = cloneref(part)

-- The wrappers themselves are not equal
assert(part ~= clone, "Clone should not be equal to the original reference wrapper")

-- But they modify the same underlying object
clone.Name = "Test"
assert(part.Name == "Test", "Clone should have updated the original instance's properties")
```

</CodeGroup>

<Tip>
  `cloneref` is commonly used to bypass anti-cheat scripts that scan game directories for unrecognized or hook-modified references to important service objects.
</Tip>
