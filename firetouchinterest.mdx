```luau
firetouchinterest(part: BasePart, transmitter: BasePart, toggle: boolean | number): ()
```

`toggle{:luau}` decides which signal fires. `0{:luau}` or `true{:luau}` raises `.Touched{:luau}`, as if the parts just made contact. `1{:luau}` or `false{:luau}` raises `.TouchEnded{:luau}`, as if they just separated.

## Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `part` | `BasePart` | The first part in the simulated touch. |
| `transmitter` | `BasePart` | The second part in the simulated touch. |
| `toggle` | `boolean \| number` | `0{:luau}` or `true{:luau}` for touch start; `1{:luau}` or `false{:luau}` for touch end. |

## Returns

<code title="the function does not return a value.">void</code>
