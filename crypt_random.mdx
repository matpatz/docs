---
title: "crypt.random"
---

```luau
crypt.random(size: number): string
```

Each call returns a different result. For random bytes already Base64-encoded use `crypt.generatebytes{:luau}`; for an AES key use `crypt.generatekey{:luau}`.

Most commonly used as a raw-randomness source for tokens, salts, and nonces.

## Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `size` | `number` | Number of random bytes to return. |

## Returns

`string` - `size{:luau}` random bytes.

## Example
```
