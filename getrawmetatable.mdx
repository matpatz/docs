---
title: "getrawmetatable"
---

```luau
getrawmetatable(object: table | userdata): table?
```

If the `__metatable{:luau}` metamethod is set, `getmetatable{:luau}` returns that value instead of the actual metatable itself. `getrawmetatable{:luau}` skips that lock, and returns the metatable directly.

Most commonly used to grab the `game{:luau}`'s `__index{:luau}` or `__namecall{:luau}` metamethods so you're able to hook them.

## Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `object` | `table \| userdata` | The table or userdata to inspect. |

## Returns

`table?` - the metatable of `object{:luau}`, or `nil{:luau}` if it has none. The returned table is the source's metatable - **not a copy**. Editing the metatable mutates the live source.

```luau

```

## Example

<CodeGroup>

```luau Example 1
getrawmetatable(object: table | userdata): table?
```

```luau Example 2
print(getmetatable(game)) --> "This metatable is locked!"
print(getrawmetatable(game)) --> table: 0x... 
                             --- which is the actual metatable
```

</CodeGroup>

<Tip>
